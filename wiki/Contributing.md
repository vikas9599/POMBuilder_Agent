# Contributing

Thanks for your interest in contributing! A few simple guidelines:

- Fork the repository and open a pull request against `main`.
- Follow existing code style and include tests where applicable.
- Update `README.md` and relevant `wiki/` pages when adding or changing features.
- For significant changes, open an issue first to discuss the design.

Code of conduct

Please be respectful and follow friendly collaboration norms when interacting in issues and PRs.
