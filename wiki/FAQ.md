# FAQ

Q: What input files are supported?
A: Playwright `.js` and `.ts` test files that contain test scenarios are supported. Keep them focused to one test or spec per upload.

Q: Does the tool run tests or just refactor code?
A: It only analyzes and refactors test script source into the POM structure; it doesn't execute tests by itself.
